mkdir diario-estudos
cd diario-estudos
git init

🥈 2️⃣ Criando o arquivo
touch estudos.txt
git add .
git commit -m "Criando arquivo de diário de estudos"

🌿 DESAFIO EXTRA — Branch

Criando nova branch:

git branch aprendizado-branch
git checkout aprendizado-branch

ou versão moderna:

git checkout -b aprendizado-branch

Agora adicione no arquivo:

Estudei sobre branch e merge.

Depois:

git add .
git commit -m "Estudos sobre branch"
🔁 Voltando para main
git checkout main
🔀 Fazendo merge
git merge aprendizado-branch

Saída:

Updating ...
Fast-forward
 estudos.txt | 1 +
📊 Conferindo tudo
git log --oneline --graph

Você verá algo assim:

* 9f8g7h Estudos sobre branch
* d4e5f6g Adicionando estudos do Dia 3
* c3d4e5f Adicionando estudos do Dia 2
* b2c3d4e Adicionando estudos do Dia 1
* a1b2c3d Criando arquivo de diário de estudos

🔥 Agora você já está usando Git como desenvolvedor de verdade.

🏆 Se quiser deixar nível profissional

Depois disso você pode:

git remote add origin https://github.com/seuusuario/diario-estudos.git
git push -u origin main
